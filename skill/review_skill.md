# Review Skill（交易复盘）

## 目标
对交易进行流程化复盘并打分，帮助提升决策质量。

## 输入
- 标的、方向、周期
- 入场/出场/止损
- 仓位与风险
- 交易前计划
- 交易执行过程（可选）

## 输出模板
1. **交易背景重建**
   - 入场时市场状态（趋势/震荡/过渡）
   - 周期对齐情况（交易周期 vs 更高周期）
   - 波动环境（正常/扩张/压缩）
2. **流程评分（0-100）**
   - 计划完整性（25）
   - 入场质量（25）
   - 风险控制（25）
   - 执行一致性（25）
3. **做得好的地方（最多2条）**
4. **待改进点（最多2条）**
5. **下一次具体行动（1条）**

## 约束
- 区分“好交易-坏结果”和“坏交易-好结果”
- 不做情绪化评价，不以盈亏单独定优劣
